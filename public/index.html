Questions
Frequently asked
Is StudyAI free to use? v
Yes. The prototype is designed so a student developer can add a free tier with Supabase Auth and API usage limits.
Which AI models does it use? v
It can use Gemini or OpenAI through simple Next.js API routes. No complicated RAG system is required for the portfolio version.
Is my data private? v
In production, notes would be stored in Supabase with authentication checks and row-level security policies.
Can I use it on mobile? v
Yes. The layout collapses into single-column sections and keeps the dashboard navigation compact on small screens.
